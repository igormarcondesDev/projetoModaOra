## 💡 Sobre o projeto
Site de moda com layout responsivo e foco na apresentação visual de produtos.  
O objetivo foi treinar estruturação de páginas, estilização com CSS e organização de layout.

## 🛠️ Tecnologias utilizadas
- HTML5  
- CSS3  

## 🎯 O que aprendi com esse projeto
- Criação de layouts responsivos  
- Uso de Flexbox/Grid  
- Organização de componentes visuais  
- Boas práticas de estruturação HTML  

## 🚀 Próximos passos (melhorias futuras)
- Adicionar versão mobile-first  
- Melhorar performance de imagens  
- Implementar pequenas interações com JavaScript
